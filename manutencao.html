Seu Windows 11 está lento? Vamos resolver.

Antes de pensar em formatar, siga este guia de manutenção preventiva. Muitos problemas podem ser corrigidos sem perder seus arquivos.

📥 Baixar Script Completo (.bat)
Baseado em mais de 15 anos de experiência com suporte técnico Windows.

📋 Como usar este guia

  1. Abra o CMD como Administrador: pressione Win + S, digite cmd, clique com o botão direito e escolha Executar como Administrador.
  2. Siga os passos na ordem abaixo. Cada comando tem um tempo estimado.
  3. Use o script automático para executar todos os passos de uma vez (recomendado).
Win + S → digite cmd → Executar como Administrador

🛡️ Os 8 passos da manutenção preventiva

Passo 1
🛡️
CHKDSK — Verificação de disco
Escaneia e repara erros no disco rígido, setores danificados e problemas de sistema de arquivos.
chkdsk /f /r
⏱ Tempo estimado: 30-60 min
Passo 2
🔧
SFC — Verificador de arquivos do sistema
Verifica e restaura arquivos protegidos do Windows substituindo versões corrompidas.
sfc /scannow
⏱ Tempo estimado: 10-30 min
Passo 3
⚙️
DISM — Reparo da imagem do Windows
Corrige a imagem de sistema corrompida que pode impedir o SFC de funcionar.
DISM /Online /Cleanup-Image /RestoreHealth
⏱ Tempo estimado: 15-30 min
Passo 4
🧹
Limpeza de arquivos temporários
Remove arquivos temporários desnecessários que acumulam e ocupam espaço.
del /q /f /s %TEMP%\* & rd /s /q %TEMP%
⏱ Tempo estimado: 1 min
Passo 5
🗑️
CleanMgr — Limpeza profunda
Ferramenta nativa que libera espaço removendo arquivos de sistema, caches e atualizações antigas.
cleanmgr /sagerun:1
⏱ Tempo estimado: 5-10 min
Passo 6
🔄
Boot Repair — Reparo de inicialização
Reconstrói o gerenciador de inicialização e corrige problemas de boot.
bootrec /fixboot & bootrec /fixmbr
⏱ Tempo estimado: 2 min
Passo 7
⚡
Desfragmentação (HDD)
Reorganiza arquivos no disco rígido para melhorar desempenho (apenas para HDD, ignore se for SSD).
defrag C: /O
⏱ Tempo estimado: 10-30 min
Passo 8
🌐
Redefinição de Rede
Limpa cache DNS, reseta winsock e pilha TCP/IP para corrigir problemas de conectividade.
ipconfig /flushdns & netsh winsock reset & netsh int ip reset
⏱ Tempo estimado: 1 min

✅ Checklist de Execução

⚠️ Quando a manutenção não resolve — é hora de formatar

CHKDSK encontra setores ruins repetidamente
🔴 Disco morrendo. Substitua o HD/SSD imediatamente.
SFC não repara mesmo após DISM
🔴 Sistema muito corrompido. A formatação é o caminho mais seguro.
PC continua lento após todos os passos
🔴 Pode ser hardware defasado ou software malicioso. Uma instalação limpa resolve.
Tela azul persistente (BSOD)
🔴 Erro crítico que geralmente indica problema de driver ou corrupção grave. Formate.
