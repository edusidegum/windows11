🛡️ Manutenção Preventiva do Windows 11

Antes de formatar, siga este guia. Muitos problemas podem ser corrigidos sem perder seus arquivos. Execute cada comando manualmente para maior segurança.

🔍

1. CHKDSK

Verifica e corrige erros no disco rígido, setores danificados e problemas de integridade do sistema de arquivos.

chkdsk C: /f /r
⏱️ 10–30 min
🛠️

2. SFC

Verifica a integridade de todos os arquivos protegidos do sistema e substitui arquivos corrompidos por versões originais.

sfc /scannow
⏱️ 15–30 min
⚙️

3. DISM

Repara a imagem do sistema Windows, corrigindo corrupções no repositório de componentes e restaurando a saúde do sistema.

DISM /Online /Cleanup-Image /RestoreHealth
⏱️ 20–40 min
🧹

4. Limpeza de Temporários

Remove arquivos temporários do sistema, cache de atualizações e lixo digital que acumulam espaço e lentidão.

del /q/f/s %TEMP%\* && del /q/f/s C:\Windows\Temp\*
⏱️ 2–5 min
🗑️

5. CleanMgr

Ferramenta nativa de limpeza de disco que libera espaço removendo arquivos desnecessários, incluindo atualizações antigas.

cleanmgr /sageset:1 && cleanmgr /sagerun:1
⏱️ 5–15 min
🔧

6. Boot Repair

Reconstrói o registro de inicialização e repara problemas de boot, como tela preta ou loop de reinicialização.

bootrec /fixmbr && bootrec /fixboot && bootrec /rebuildbcd
⏱️ 5–10 min
💿

7. Desfragmentação

Otimiza a disposição dos dados no disco rígido (HDD) para melhorar o desempenho de leitura e escrita.

defrag C: /O
⏱️ 30–120 min (HDD)
🌐

8. Rede

Reseta pilha TCP/IP, libera e renova IP, e limpa cache DNS para solucionar problemas de conexão.

netsh int ip reset && ipconfig /release && ipconfig /renew && ipconfig /flushdns
⏱️ 2–5 min

📥 Executar Manutenção

⭐ Método Manual — Recomendado

A execução manual é mais segura porque você vê o resultado de cada comando antes de prosseguir. Siga os passos: Win+R → digite cmd → clique com botão direito e escolha Executar como administrador → digite cada comando abaixo e pressione Enter.

1. CHKDSK — Verificação de Disco
chkdsk C: /f /r
Verifica e repara erros no disco. Pode exigir reinicialização.
⏱️ 10–30 min
2. SFC — Verificador de Arquivos do Sistema
sfc /scannow
Escaneia e restaura arquivos protegidos do sistema.
⏱️ 15–30 min
3. DISM — Reparo da Imagem do Windows
DISM /Online /Cleanup-Image /RestoreHealth
Repara a imagem do sistema operacional.
⏱️ 20–40 min
4. Limpeza de Arquivos Temporários
del /q/f/s %TEMP%\* && del /q/f/s C:\Windows\Temp\*
Remove arquivos temporários do usuário e do sistema.
⏱️ 2–5 min
5. CleanMgr — Limpeza de Disco
cleanmgr /sageset:1 && cleanmgr /sagerun:1
Abre a ferramenta de limpeza com configurações pré-definidas.
⏱️ 5–15 min
6. Reparo de Inicialização (Boot)
bootrec /fixmbr && bootrec /fixboot && bootrec /rebuildbcd
Reconstrói o registro de boot do Windows.
⏱️ 5–10 min
7. Desfragmentação (HDD)
defrag C: /O
Otimiza o disco rígido (HDD). Em SSDs, o Windows otimiza automaticamente.
⏱️ 30–120 min (HDD)
8. Reset de Rede
netsh int ip reset && ipconfig /release && ipconfig /renew && ipconfig /flushdns
Reseta pilha TCP/IP, renova IP e limpa cache DNS.
⏱️ 2–5 min
⚠️ Recomendação: Sempre execute manualmente cada comando para acompanhar os resultados. O script automático é uma alternativa prática, mas não substitui a supervisão manual.

🚨 Quando Formatar?

Se estes sinais persistirem após a manutenção, considere formatar.

🐌

Lentidão extrema

O sistema demora mais de 5 minutos para iniciar mesmo após limpeza.

💥

Erros frequentes

Telas azuis (BSOD) ou travamentos aleatórios que não desaparecem com reparos.

🔐

Infecção persistente

Malware ou vírus que não são removidos mesmo com scanners avançados.

📦

Atualizações falham

Windows Update não consegue instalar atualizações mesmo após DISM e SFC.
